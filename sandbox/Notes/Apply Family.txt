base::apply             Apply Functions Over Array Margins

  apply(array, margin, function, ...)
    Margin is the dimension number to apply over.  If array is a matrix, then 1 is rows and 2 is columns
    Array is the data
    Function is the function to apply
    ... is a list of additional parameters to pass through to the function
